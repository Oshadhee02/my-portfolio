LabMaster

2D Virtual Lab for A/L Students

Lab Master is a Unity-based 2D platform for Sri Lankan A/L students to conduct practicals virtually. Includes features like login/signup, quiz section, note making, and teacher guidance.

  • 🔧 Built with Unity + WebGL
  • 📦 Backend: Node.js + MongoDB
  • ☁️ Hosted on Vercel
